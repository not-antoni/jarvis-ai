# Contributing

Thanks for helping out! To keep things simple and legal, by committing code or submitting a Pull Request, you agree that:

* **You own the code:** It’s yours to give and not stolen.
* **I can use it:** You grant me a permanent, free license to use, modify, and even re-license this project (including your work) if needed.
* **It stays merged:** Contributions are permanent so the project doesn't break later.

**Got concerns?** If you have any questions about these terms or how we’re handling the project's future, just message me directly or open an issue. Let's build something cool!
