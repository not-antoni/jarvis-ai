# Jarvis Discord Bot Setup Guide

## Bot Invitation Link
To invite Jarvis to your Discord server, you'll need to:

1. Go to https://discord.com/developers/applications
2. Select your "Jarvis" application
3. Go to OAuth2 > URL Generator
4. Select scopes: `bot` and `applications.commands`
5. Select bot permissions:
   - Send Messages
   - Use Slash Commands
   - Read Message History
   - Add Reactions
   - Embed Links
   - Attach Files
   - Read Messages/View Channels
   - Use External Emojis

6. Copy the generated URL and open it to invite Jarvis to your server

## Bot Features

### Core Personality
- Sophisticated and articulate (like Marvel's Jarvis)
- Helpful with subtle wit and dry humor
- Remembers conversations and learns from interactions
- Protective and respectful to users

### How to Interact
- Mention @Jarvis in any channel
- Send a direct message
- Just say "Jarvis" in a message

### Advanced Features
- **Memory System**: Remembers past conversations and user preferences
- **Autonomous Thoughts**: Occasionally shares insights or observations
- **Learning**: Adapts responses based on interaction history
- **Evolution**: Becomes more sophisticated over time

## Commands
Jarvis doesn't use traditional commands - just talk to him naturally!

Examples:
- "Jarvis, what's the weather like?"
- "Hey Jarvis, can you help me with something?"
- "Jarvis, what do you think about..."
